List Exercises

It is possible see all exercises registered in the system (available or not) by the URL http://dataccc.gmf.ufcg.edu.br/hoopaloo/exercises or by Exercise > List Exercises menu option (1).

In this page are showed all exercises registered in the system and the informations of those exercises (name, description, availability, owner, date of creation, date of will be unavailable, number of tests registered for that exercise, number of students that solved it, mean of notes of students for that exercise, the option of change the availability (4) and the option of delete it (5)). Each line of table (2) is a exercise.

The exercise name is a link to exercise page (3), in which there is the option of edit it, and if exists submitions, is possible see the a summary of results (See topic Exercise View).

Only the teacher and the assistants can see the details of an exercise.
