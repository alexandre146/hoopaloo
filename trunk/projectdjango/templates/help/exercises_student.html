Exercises of a student

To see at this page, you must click at a link in General Students View. This page is a view with details about all exercises delivered by a student.

The details about the exercises are showed in a table, in which each line is a different exercise. The columns contains informations as: the name of exercise (a link to see details of submission - code, comments, note), the number of submission realized by this student, the date of last submission, the number of tests registered to this exercise, the number of errors of student's code (result of test execution), the note of student and the percentage of right of the class.

Only the teacher and the assistants can see details of students' exercises.
