General Students View

This page can be accessed by URL http://dataccc.gmf.ufcg.edu.br/hoopaloo_test/teacher_view or by Views > General Students View menu option (1). At this page is possible see a table with some informations of students (2), in this table each line is a student.

In this table you can see name, assistant's username, number total of submissions of this student, number of exercises available that this student not submit, number of undelivered exercises, rate of submissions number by exercises number, mean of notes and a link to page of exercises of this student (4).

At the student's name exists a link (3) to a page with details of delivered exercises of that student.

Only the teacher and the assistants can see details of students.
