New Exercise

The teacher and the assistants can register new exercises in the system and, if it was available, the students can submit solutions for its. This register can be made by URL http://dataccc.gmf.ufcg.edu.br/hoopaloo_test/add_exercise or by Exercise > New Exercise menu option (1).

To register a new exercise is necessary informs: the exercise name/title (this name must be unique)(2), a description of this exercise (3), if the exercise is available since its creation (checkbox marked) or not (checkbox unmarked) (4) and the date in which the exercise should be unavailable (5).

All fields are required. If you not filled all fields an error will appear indicating this problem. If not exists any error, the following message will be showed:

Only the teacher and the assistants can add exercises.
