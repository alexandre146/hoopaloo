Submission View

Clicking at the link in the name of exercise in Exercises of a Student page you will redirect to Submission View page. Here, you will see some details of submission, for example, the code of student (1). Here, too, is possible do comments (3) about the code and register a note (4). Additionally, you can see the code of test (2) related to the exercise.

Only the teacher and the assistants can see details of students' submission.
